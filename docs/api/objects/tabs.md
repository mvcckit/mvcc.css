#Tabs Object

The `tabs` object creates equal-width navigation from a list.

	<ul class="o-tabs">
		<li><a href="#"></a></li>
		<li><a href="#"></a></li>
		<li><a href="#"></a></li>
	</ul>

## Options

**tabs--xs**

The `tabs--xs` option applies an extra-small padding around tabs.

	<ul class="o-tabs o-tabs--xs">
		<li><a href="#"></a></li>
		<li><a href="#"></a></li>
		<li><a href="#"></a></li>
	</ul>
